Getting Started
-----------------------

1. Pull the repo and run GenerateSolution.bat
2. Run Engine/ThirdParty/GenerateSolutions.bat (Until I figure out a good way to remove this step)
3. Open the generated solution in the root folder
4. Build & Celebrate
